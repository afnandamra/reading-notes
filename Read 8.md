# Operators and Loops

### Comparison Operators:
* == : equal to (same value).
* != : not equal to.
* === : strict equal (same data type and value).
* !== : strict not equal to.
* \> : greater than.
* < : less than.
* \>= : Greater than or equal to.
* <= : less than or equal to.

### Logical Operators:
* && : logical and.
* \|\| : logical or.
* ! : logical not.

---
### for loop:
```JavaScript
    for (var a=0; a < 10; a++){
        document.write(a);
    }
```

### While loop:
```JavaScript
    while (a < 10){
        document.write(a);
        a++;
    }
```
