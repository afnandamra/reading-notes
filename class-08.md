# CSS
## Layout

` <container>` is used to contain elements.
` <div> ` is used to group elements to a section.


``` css
position: static;
position: absolute;
position: relative;
position: fixed;
z-index: 10px; /* fixed elements (overlapping, sticky) */
float: right;
width: 500px;
height: 500px;
clear: left;
overflow: auto;
```
